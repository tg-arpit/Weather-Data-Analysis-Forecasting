# 🌍 Cities & Countries Analysis Dashboard (Power BI)

## 📌 Project Overview

This project is an interactive **Power BI Dashboard** that analyzes global cities and countries using geographical and population data. The dashboard provides insights into continents, countries, cities, population statistics, and regional distribution through interactive visualizations.

The goal of this project is to demonstrate data visualization, business intelligence, and dashboard development skills using Microsoft Power BI.

---

## 🚀 Features

* Interactive Power BI Dashboard
* KPI Cards displaying:

  * Total Population
  * Total Continents
  * Total Countries
  * Total Cities
  * Population Density
* Map visualization of cities using latitude and longitude
* City distribution by country
* Country distribution by continent
* Regional analysis of countries
* Interactive slicers for filtering data

---

## 📊 Dashboard Visualizations

* KPI Cards
* Clustered Column Charts

* Interactive Map
* Filters & Slicers

## ▶️ How to Use

1. Clone this repository.
2. Open the `.pbix` file using **Microsoft Power BI Desktop**.
3. Refresh the data if necessary.
4. Explore the dashboard using the available filters and slicers.
